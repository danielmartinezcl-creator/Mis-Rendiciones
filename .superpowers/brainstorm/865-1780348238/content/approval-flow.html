¿Cómo funciona la aprobación?

Este flujo define la experiencia central del aprobador. Elegí el modelo que mejor refleja cómo trabaja tu empresa.

OPCIÓN A — SIMPLE

Un clic, todo aprobado

El aprobador ve el resumen y aprueba o rechaza la rendición completa.

Rendición #042 — Juan Pérez
Taxi aeropuerto$18.500
Almuerzo cliente$32.000
Hotel Santiago$95.000
Total $145.500
✓ Aprobar todo
✗ Rechazar

✓ Pros

  • Rapidísimo para el aprobador
  • UX mínima

✗ Contras

  • No permite rechazar un solo ítem
  • Aprobador pierde control fino
OPCIÓN B — ÍTEM POR ÍTEM RECOMENDADO

Control granular por ítem

El aprobador puede aprobar o rechazar cada gasto individualmente, con motivo de rechazo.

OPCIÓN C — MULTI-NIVEL

Cadena de aprobación

Supervisor → Gerente → Finanzas. Montos bajos saltan niveles automáticamente.

Empleado envía
→
Supervisor
→
Gerente
↓ (si monto > $100.000)
Finanzas aprueba
→
Reembolso
Auto-aprobación si monto < $50.000 y categoría preaprobada (taxi, colación, etc.)

✓ Pros

  • Control corporativo total
  • Auto-aprobación reduce carga

✗ Contras

  • Complejidad alta en v1
  • Configuración por empresa difícil
