Diseño — Sección 1: Roles y vistas del sistema

Tres tipos de usuario, tres experiencias distintas. Cada rol tiene su propio dashboard y permisos.

👤
Rendidor
Empleado que gasta
PUEDE HACER
✓ Crear rendición nueva
✓ Subir foto de boleta/factura
✓ OCR automático del documento
✓ Agregar N ítems por rendición
✓ Enviar a aprobación
✓ Ver estado de sus rendiciones
✓ Ver historial de reembolsos
✗ Ver rendiciones de otros
✗ Aprobar
Vista principal: móvil-first, captura rápida con cámara, flujo de 3 pasos.
✅
Aprobador
Jefe / supervisor
PUEDE HACER
✓ Ver rendiciones de su equipo
✓ Aprobar ítem por ítem
✓ Rechazar con motivo por ítem
✓ Aprobar rendición completa
✓ Ver fotos de respaldo
✓ Devolver a borrador
✓ También puede ser rendidor
✗ Marcar como reembolsado
✗ Gestionar usuarios
Vista principal: bandeja de entrada estilo email, pendientes primero, desktop + móvil.
🏢
Administrador
Contabilidad / finanzas
PUEDE HACER
✓ Todo lo del aprobador
✓ Ver todas las rendiciones
✓ Marcar reembolso como pagado
✓ Dashboard financiero
✓ Exportar Excel / PDF
✓ Gestionar empleados
✓ Configurar aprobadores
✓ Gestionar categorías
✓ Gestionar organización
Vista principal: dashboard con KPIs, tabla maestra de todas las rendiciones, filtros avanzados.
